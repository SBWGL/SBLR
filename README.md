***use SpringBoot + thymeleaf + JPA + SpringSecurity + OAuth2 + Bootstrap + jQuary***

	- 객실 보기
	- 객실 상세보기
	- 객실 추가
	- 객실 수정
	- 객실 검색
	- 페이징
	- 리뷰

- 결제
	- 아임포트의 카카오페이 결제 서비스
  
- 예약 확인 페이지
	- 예약 정보 보기
  
- 나와 접촉 페이지
	- Google Maps를 통한 지도
